# RNA Secondary Structure Prediction

## Introduction

This repository contains a presentation and implementation of a method for predicting RNA secondary structures using Stochastic Context-Free Grammar (SCFG) and Evolutionary History.

## Contents

1. **Presentation**: A comprehensive presentation explaining the theory and methodology behind the prediction of RNA secondary structures using SCFG and Evolutionary History.

2. **Implementation**: Source code for the implementation of the method described in the presentation.
